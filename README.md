# Pomodoro Timer

A minimalist Pomodoro timer web app with dark glassmorphism styling, responsive layout, and ambient audio options.

## Features

- Work / short break / long break mode switching
- Editable timer durations in settings
- Ambient sound options (lofi, rain, bonfire, lofi beats)
- Finish sound and subtle beep notifications
- Local settings persistence using `localStorage`
- Responsive portrait and landscape layout

## Usage

1. Open `index.html` in a browser.
2. Use the mode buttons to select `Pomodoro`, `Short Break`, or `Long Break`.
3. Press `Start` to begin the countdown.
4. Open `Settings` to adjust durations, volume, and sound behavior.

## Files

- `index.html` — app structure and UI
- `styles.css` — dark theme and responsive styling
- `script.js` — timer logic, audio, and settings persistence

## Notes

For best results, use a modern browser with Web Audio API support.
